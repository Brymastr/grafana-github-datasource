Set up a GitHub application

  • For standard GitHub, click here to go to applications settings in a new window.

  • Click "Register new application" and fill out the form:

    • Homepage URL: https://<orgname>.grafana.com/
    • Application Name: Anything you like, e.g. Grafana
    • Application description: Anything you like, optional
    • Authorization callback URL: https://localhost:3443/

  • Click "Register Application"

GitHub details

Copy and paste the Client ID and Secret from the upper-right corner of your newly-created application.
